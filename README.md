# Hurricane Tracks · Dominican Republic (1851–2025)

An interactive, single-file web visualization of historical hurricane tracks affecting the Dominican Republic from 1851 to 2025.

## 🌐 Live site

Once GitHub Pages is enabled, the site is available at:

```
https://<your-username>.github.io/hurricane-tracker-dr/
```

## 📂 About

This is a fully self-contained `index.html` — all HTML, CSS, and JavaScript are in one file, with no external build step or dependencies required. Just open it in any modern browser.

## 🚀 Running locally

Simply open `index.html` in your browser, or serve it with any static server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
